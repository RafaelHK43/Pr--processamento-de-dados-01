
# Pré-processamento de Dados - VA1 Individual

Este repositório contém o projeto de pré-processamento de dados desenvolvido para a primeira Verificação de Aprendizagem (VA1) da disciplina de Pré-processamento de Dados. O foco principal é a limpeza, tratamento e exploração inicial de conjuntos de dados utilizando Python e bibliotecas de análise de dados.

## 🚀 Objetivo do Projeto

O objetivo deste notebook é aplicar técnicas fundamentais de preparação de dados para garantir a qualidade das informações antes de qualquer etapa de modelagem ou visualização avançada.

As etapas contempladas incluem:
- Identificação e tratamento de valores ausentes (NaN).
- Remoção de duplicatas.
- Normalização e padronização de escalas.
- Codificação de variáveis categóricas.
- Análise estatística descritiva básica.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido em ambiente Jupyter Notebook utilizando as seguintes ferramentas:

* **Python 3**
* **Pandas**: Manipulação e análise de dados.
* **NumPy**: Suporte para arrays e cálculos matemáticos.
* **Matplotlib / Seaborn**: Visualização de dados e gráficos estatísticos.
* **Scikit-Learn**: Utilizado para técnicas de escalonamento e pré-processamento.

## 📁 Estrutura do Repositório

- `VA1_Individual_Pré_processamento_de_dados.ipynb`: Notebook principal com todo o código-fonte e documentação das etapas realizadas.
- `README.md`: Documentação do projeto.

## ⚙️ Como executar o projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/RafaelHK43/Pr--processamento-de-dados-01.git](https://github.com/RafaelHK43/Pr--processamento-de-dados-01.git)
    ```
2.  **Navegue até a pasta:**
    ```bash
    cd Pr--processamento-de-dados-01
    ```
3.  **Instale as dependências necessárias:**
    ```bash
    pip install pandas numpy matplotlib seaborn scikit-learn
    ```
4.  **Inicie o Jupyter Notebook:**
    ```bash
    jupyter notebook
    ```
5.  Abra o arquivo `.ipynb` e execute as células sequencialmente.
